# Gehenna
[![Build Status](https://travis-ci.org/beryll1um/Gehenna.svg?branch=stable)](https://travis-ci.org/beryll1um/Gehenna.svg?branch=stable)

## Task
Create a backend of the service for chatting with asymmetric encryption.
